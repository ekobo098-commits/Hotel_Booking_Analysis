🏨 Hotel Booking Analysis | SQL + Power BI
📌 Project Overview

This project analyzes hotel booking data using PostgreSQL for data exploration and Power BI for interactive visualization. The goal is to uncover booking patterns, customer behavior, cancellations, and estimated revenue to support business decision-making.

📌 Key Business Insights
Online Travel Agencies generated the highest number of bookings.
Transient customers contributed the largest share of bookings and estimated revenue.
City Hotels generated higher estimated revenue than Resort Hotels.
Cancellation rates varied significantly across market segments.
Only a small percentage of guests were repeat customers, suggesting opportunities for customer retention strategies.
